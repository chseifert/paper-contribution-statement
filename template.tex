\subsubsection*{Legend}
Contributions are categorized into low medium and substantial.\footnote{For scientific work, measuring contributions exactly is impossible, therefore a broad categorization scheme is used.} The final assessment is based on self-assessment and discussion among authors. Some CRediT categories might not apply to each paper (e.g., if no software is produced or no funding was acquired), this is indicated with ---~n.a.~---. Some papers require a lot of work in one category, and minor work in another. Thus, multiple authors can contribute substantially to one aspect, and rows visually not necessarily add up to the same value.
